# Contributing to Prompt Weaver

Thank you for your interest in contributing to Prompt Weaver! We welcome contributions from the community.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Install dependencies**:
    ```bash
    pnpm install
    ```
4.  **Create a new branch** for your feature or bugfix.
5.  **Make your changes**.
6.  **Run tests** to ensure everything is working:
    ```bash
    pnpm test
    ```
7.  **Commit your changes** and push to your fork.
8.  **Submit a Pull Request** to the main repository.

## Development

- **Build**: `pnpm build`
- **Test**: `pnpm test`
- **Lint**: `pnpm lint`
- **Format**: `pnpm format`

## Code Style

We use [Biome](https://biomejs.dev/) for linting and formatting. Please ensure your code passes the linting checks before submitting a PR.

## Questions?

If you have any questions, feel free to open an issue or discussion in the repository.
